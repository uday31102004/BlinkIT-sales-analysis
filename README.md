 🛒 Blinkit Sales Analysis Dashboard
An interactive Excel-based Sales Analysis Dashboard built using Pivot Tables, Power Query, and data visualization techniques to analyze Blinkit grocery sales performance across multiple business dimensions.

Project Overview
This project focuses on analyzing Blinkit grocery sales data to generate meaningful business insights.  

The dashboard provides a clear view of sales performance, product distribution, outlet performance, and customer trends.

The objective of this project is to demonstrate practical data analysis and dashboard-building skills using Microsoft Excel.


Key Features
 KPI Cards (Total Sales, Average Sales, Number of Items, Average Rating),
 Sales Trend by Outlet Establishment Year, 
 Outlet Size Performance Analysis, 
 Outlet Location Comparison (Tier 1, Tier 2, Tier 3), 
 Item Type Contribution Analysis, 
 Fat Content Distribution, 
 Outlet Type Analysis (Total Sales, Avg Sales, Item Count), 
 Interactive Filters using Slicers.


Tools & Technologies Used
 Microsoft Excel, 
 Pivot Tables, 
 Pivot Charts, 
 Slicers, 
 Power Query (Data Cleaning & Transformation), 
 Data Modeling, 
 Conditional Formatting.


Dataset
The dataset contains information about:
 Item Type, 
 Fat Content, 
 Outlet Size, 
 Outlet Location Type, 
 Outlet Establishment Year, 
 Sales, 
 Ratings.


Insights Derived:

Tier 3 outlets generate the highest total sales.

Regular fat products contribute more revenue than low-fat items. 

Supermarket Type 1 has the highest item count. 

Sales peaked around 2018 based on outlet establishment trend.


Learning Outcomes:

 Hands-on experience in building interactive dashboards
 
 Strong understanding of Pivot Tables and aggregation techniques
 
 Practical use of Power Query for data cleaning 
 
 Ability to extract business insights from raw data
 
 Dashboard design and layout structuring
